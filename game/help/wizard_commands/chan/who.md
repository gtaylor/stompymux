# @chan/who

List a channel's active members, or all membership records:

```text
@chan/who <channel>
@chan/who <channel>/all
```

The output identifies each member as a player or object and shows whether the
member is listening. Hidden-member visibility follows the channel's
transparent setting and the viewer's wizard privileges.
