# Wizard-only help file

This help file should only show up in indexes for and be readable by wizards.

Native control is role-only. God can control every object and player. Wizards
can control every non-Wizard object and player, but cannot control themselves,
God, or another Wizard. Mortals cannot control objects, including themselves.
Zones do not grant control. Trusted Lua logic can inspect and manipulate every
object through the `mux` API.
