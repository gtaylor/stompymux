# XCODE (`X`)

Marks an object as a BattleTech special object managed by the native XCode
subsystem. Only Wizards or God may set it. Once set, only God may clear it
through the normal flag command.

`XCODE` and `ANSI` share the compact display letter `X`; use the full flag name
when changing either flag.
